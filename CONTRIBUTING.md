# Contributing to Investment DNA

Thanks for your interest in contributing. This project aims to be the standard open-source framework for capturing institutional investment firm context for AI systems.

## How to Contribute

**New templates or sections** — If your firm tracks something not covered by the existing templates (e.g., co-investment tracking, LP relationship management, ESG scoring), open an issue describing the use case, then submit a PR with the template and an example.

**Vertical adaptations** — The framework currently covers PE, credit, and CRE. If you work in venture, infrastructure, real assets, or another strategy, we'd love vertical-specific guidance added to `VERTICALS.md` or as new template variants.

**Evaluation improvements** — Better scorecard weights, new evaluation prompts, or integration guides for specific AI platforms (Claude Projects, ChatGPT custom GPTs, etc.).

**Examples** — Additional example firms across different strategies, fund sizes, and stages. Fictional firms only — do not submit real firm data.

**Bug fixes and clarity** — Typos, unclear template guidance, missing fields, or structural improvements.

## Guidelines

- Keep everything in markdown
- Templates should include inline comments (`<!-- -->`) explaining what to fill in and why
- Examples should be realistic and internally consistent (portfolio positions should match stated criteria, etc.)
- Follow existing file naming conventions
- One PR per logical change

## Code of Conduct

Be professional. This is a tool for investment professionals — contributions should reflect that standard.
